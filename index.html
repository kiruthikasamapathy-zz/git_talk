Magic solution


  • Version control is a system that records changes to a file or set of files over time so that you can recall specific versions later

Version control systems

git


  • Open source
  • Distributed
  • Designed for speed and efficiency

Everything is local (almost)


  • clone a repo, not checkout
  • complete copy of repository, not just the working copy
  • users have their own repository in the local machine
  • only when you have to share your work with the world, then you need to connect to the remote machine

  • stores the snapshot of all files rather than delta changes
  • like a mini file system
